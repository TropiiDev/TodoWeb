How to use

Here is how to use the example todo list!

  • Click on the "Edit" button to change the name of the todo.
  • Clicking on the checkbox will delete the todo
  • Clicking on the title of the allow you to edit the todo
  • Clicking the Add button will allow you to add a new todo
  • Finally, clicking on the Save button will save all changes to the todo list.
